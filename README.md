# React Chat App

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

## Installation

Use the package manager [npm](https://www.npmjs.com/) to install dependencies in the root directory.

```bash
npm install
or yarn install
```

To start developing

```bash
npm run start
or yarn start
```

To start building

```bash
npm run build
```

To start testing

```bash
npm run test
```
